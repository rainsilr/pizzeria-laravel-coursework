# История изменений

В этом файле фиксируются основные изменения учебного проекта «Пиццерия Laravel».

## В разработке

- Подключение HTTPS и доменного имени.
- Добавление ролей администратора и клиента.
- Реализация корзины и оформления заказов.
- Загрузка изображений пицц.
- Резервное копирование базы данных.

## 1.0.0 — 18.06.2026

### Добавлено

- Публичная главная страница и каталог пицц.
- Детальная страница позиции меню.
- Связанные сущности категорий и пицц через Eloquent ORM.
- Создание, редактирование и удаление пицц.
- Регистрация и аутентификация через Laravel Breeze.
- Ограничение CRUD-маршрутов с помощью middleware `auth`.
- Серверная валидация пользовательского ввода.
- CSRF-защита изменяющих форм.
- Поиск по названию и описанию.
- Фильтрация пицц по категории.
- Пагинация каталога.
- Уведомления о результатах операций.
- Сидер с администратором, пятью категориями и пятнадцатью пиццами.
- Автоматические тесты основных пользовательских сценариев.
- Автоматический запуск тестов через GitHub Actions.
- Production-сборка frontend-ресурсов с помощью Vite.
- Развертывание приложения на VPS с Nginx, PHP-FPM и MySQL.
